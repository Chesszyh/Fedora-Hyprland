# 透明度设置

1. 修改全局默认透明度
如果你想改变所有窗口的默认透明度，请编辑以下文件：
文件位置: `UserConfigs/UserDecorations.conf`

```conf
1 decoration {
2     active_opacity = 1.0     # 活动窗口透明度 (0.0 - 1.0)
3     inactive_opacity = 0.9   # 非活动窗口透明度
4     fullscreen_opacity = 1.0 # 全屏窗口透明度
5     # ...
6 }
```

1.0 为完全不透明，0.0 为完全透明。

2. 针对特定软件设置透明度
如果你只想修改特定软件（如终端、浏览器）的透明度，请编辑：
文件位置: `UserConfigs/WindowRules.conf`

你可以在该文件中找到或添加 windowrulev2 规则。格式如下：

```conf
1 # windowrulev2 = opacity [活动透明度] [非活动透明度], class:^(应用类名)$
2 windowrulev2 = opacity 0.9 0.8, class:^(kitty)$
```

当前的配置中已经有很多针对 tag: 的透明度规则（如 tag:terminal*），你可以直接修改那些数值。

3. 临时快捷键切换
根据 `UserConfigs/UserKeybinds.conf`，已配置了一个快捷键来临时切换当前窗口是否透明：

* 快捷键: Super + Ctrl + O
* 功能: 在“完全不透明”和“设定透明度”之间切换。
